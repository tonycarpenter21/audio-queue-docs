🔊 Audio Channel Queue

Multi-channel audio queue management for browsers

Get Started 🚀View Live Demo 🎵
npm install audio-channel-queue
🎛️

Multi-Channel Audio

Manage multiple independent audio queues simultaneously. Perfect for games, applications, and complex audio experiences with separate channels for music, sound effects, and voice.

⏯️

Real-Time Control

Full playback control with pause, resume, volume adjustment, and priority queueing. Includes smooth fade transitions and volume ducking for professional audio experiences.

📘

TypeScript Ready

Built with TypeScript and zero dependencies. Comprehensive type definitions, extensive documentation, and a clean API make integration simple and reliable.
